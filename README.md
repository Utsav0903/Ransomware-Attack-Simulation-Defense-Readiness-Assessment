## 📌 About

RADSAS is a full-stack cybersecurity platform that simulates ransomware 
attack behaviour in a safe, sandboxed environment. It provides real-time 
monitoring, risk assessment, forensic logging, and defense readiness 
scoring — all through a modern web dashboard with a live 3D attack globe.

---

## ✨ Features

- 🌍 **Live 3D Attack Globe** — real-time attack visualization (Three.js)
- 🔐 **AES File Encryption/Decryption** — Fernet-based with 95%+ accuracy
- 📊 **Real-time Risk Scoring** — dynamic threat level assessment
- 🕵️ **Forensic Activity Logging** — complete audit trail of all actions
- 🔄 **Attack Simulation Module** — controlled ransomware behaviour
- 💾 **Recovery Module** — file restoration and rollback
- 📋 **Forensic Report Generation** — downloadable incident reports
- ⚡ **Dashboard response time under 1 second**
- 🔒 **98%+ simulation stability** across all test runs
